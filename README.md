# Roguelike Game

Minimal 2D roguelike built with TypeScript, Phaser 3, ROT.js and Vite.

## Instalacja

```
npm install
```

## Uruchomienie

```
npm run dev
```

Otwórz `http://localhost:5173`.

Repozytorium nie zawiera plików binarnych. Aby dodać własne grafiki lub dźwięki, umieść je w katalogu `assets/` i zaktualizuj kod.
Ikony PWA (`icon-192.png` oraz `icon-512.png`) należy dostarczyć ręcznie w `platform/pwa/`.

## Sterowanie

- Strzałki – ruch
- Spacja – strzał
- F – auto‑fire
- P – pauza (niezaimplementowane w MVP)
- M – wyciszenie
- F1 – pokazuj seed

npm run build
